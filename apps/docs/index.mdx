## Introduction

InBrowser is a high-performance file conversion platform that prioritizes privacy by processing files directly in the browser (WASM) or via secure server-side streams.

Designed for developers and modern web apps, InBrowser provides a simple REST API and tRPC integration to handle complex media tasks like video transcoding, PDF parsing, and document generation.
