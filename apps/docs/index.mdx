## Introduction

InBrowser is a high-performance file conversion platform that prioritizes privacy by processing files directly in the browser (WASM) or via secure server-side streams.

Designed for developers and modern web apps, InBrowser provides a simple REST API and tRPC integration to handle complex media tasks like video transcoding, PDF parsing, and document generation.

## Quick Examples

<CodeGroup>
```bash filename="gif-conversion" cURL
# Convert animated GIF to PNG
curl -X POST http://localhost:3000/api/convert \
  -F "file=@animation.gif" \
  -F "format=png"
```

```bash filename="text-conversion" cURL
# Convert markdown to image
curl -X POST http://localhost:3000/api/convert \
  -F "file=@document.md" \
  -F "format=jpg"
```

```bash filename="video-conversion" cURL
# Convert video with enhanced processing
curl -X POST http://localhost:3000/api/convert \
  -F "file=@video.webm" \
  -F "format=mp4"
```

</CodeGroup>
